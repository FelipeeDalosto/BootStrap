CONHECENDO O BOOTSTRAP

TAGS para botões

HTML

  • "button" define um botão clicavel
  • "type" define para que ira servir aquele botao
  • "type="button"" cira um botao clicavel simples
  • "type="submit"" envia informações para um banco de dados para guarda a mesma
  • "type="reset"" reseta a pagina ao estado inicial

BootStrap

  • "btn" define que é um botao
  • "btn-" sempre na frente do "btn" serve para definir o estilo do botão
  • "default" sempre na frente do "btn-" serve para feinhir o estilo
  • "btn btn-default" botão padrao
  • "btn btn-success" botão verde
  • "btn btn-primary" botão azul escuro
  • "btn btn-danger" botão vermelho
  • "btn btn-warning" botão laranja
  • "btn btn-info" botão azul claro
  • "btn btn-link" botão parecido com um link
  • "btn-block" faz com que o botão pegue todo espaço do grid
  • Podemos definir o tamnho dos botões colocando essas tegs na dentro de uma class
  • "btn-xs" extral small para telas pequenas
  • "btn-sm" small para telas de tablets
  • "btn-md" midle para telas medias
  • "btn-lg" large para telas maiores
