CONHECENDO O BOOTSTRAP


TAGS para tabs

HTML

  • "p" para paragrafo
  • "ul" lista nao ordenada
  • "ol" lista ordenada
  • "li" itens da lista
  • "a href=#" serve para colocar um link
  • dentro do "a href=#" é nescessario colocar o "ID" que ira ter o vinculo com a aba

BootStrap

  • "lead" aplica formatações para subtitulos
  • "nav" de navegação pois as tabs é uma nevegação
  • "nav nav-tabs" sintax completa para navegação em abas utilizando estilo normal
  • "nav nav-pills" outra estilização para as abas
  • "role=tablis" tem a função de ser reconhecido como uma lista de abas
  • "active" serve para dizer que o item esta ativo
  • "tab-content" onde fica os conteudos da aba
  • "tab-pane active" serve para dizer que esta aba esta referenciada por ativa
  • "tabpainel" quer dizer que este é um painel de aba
  • "data-toggle=tab" deve ser colocado em cada link para que seja conhecido como uma aba
