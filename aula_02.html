TEGS para formulario

HTML

  • "form" cria um formulario
  • input type="text" /Define uma caixa de texto simples
  • input type="password" /Define uma caixa para senha ***
  • input type="submit" /Define criar um botao
  • input type="date" /Define para criar um calendario
  • input type="radio"/Define para criar check box um opção
  • input type="checkbox" /Define para criar check
  • "label" cria uma etiqueta para utiliza no campo de formulario
  • "for" faz um vinculo entra a etiquata (nome do campo) com o input (caixa de digitação)
  • "id" cria no nome do vinculo que ira fazer

BootStrap

  • "page=header" define o cabeçalho da pagina, colocando uma linha em baixo
  • "form-grup" é o estilo de definições
  • "form-control" o estio da caixa de texto onde se preenche os formularios
