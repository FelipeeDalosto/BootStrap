Oque é BootStrap

Seria praticamente uma biblioteca com fontes, styles e css ja predefinidos, é um ferramenta que utiliza algumas TEGS esprecificas deles e tem com finalidade criar sites mais responsivos (Sites que se adaptam a difenrentes telas).

Existem varias versões do BootStrap v1.0, v2.0, v3.0, v4.0 e v5.0 porem cada tem uma documentação diferente. Todas documentações podem ser acessadas atraVEs da pagina prinipal do BootStrap.

Todas as versões podem ser acessadas neste SITE.

Configurando em um arquivo HTML.

Para configurar é bem simples, basta baixar os arquivos, ou dependendo da vesão vincular o diretamente atraz de uma API, em todas documentação tem amaneira correta de vincular dependendo da versão do BootStrap.

Sistema de Grid

Serve para montar um Layout e trabalha com 12 grids, cada grid é representado por uma coluna, cada grid pode ser divido atraves do comando "col-md-1" "col-md-12"

Opções de Grid

  • "col-xs-1" para telas menor que 768px "Extra small devices ex: Phones"
  • "col-sm-1" para telas maior que 768px "Small devices ex: Tablets
  • "col-md-1" para telas maior que 992px "Medium devices ex: Small monitor"
  • "col-lg-1" para telas maior que 1200px "Large devices ex: Large monitor"

Comandos são inserridos dentro de uma "class"

  • "container" serve para criar uma caixa que tem 12 grids.
  • "row" faz com que aquela "div" se torne uma Linha da tabela.
  • "col" serve para definir quantos grids vai ser utilizado no maximo 12 e minimo 1 na frente do "col" se coloca qual o tamanho da tela em que ele sera representado.
  • "page-header" cria um cabeçalho com formações esprecificas.

Formularios

  • "form" cria um formulario TEG de HTML
  • "label" cria uma etiqueca que serve para nomer o campo de um form
  • "for" serve para associar dois campos em relação a integração
  • "form-grup" é uma classe do bootstrap que serve para agrupar elementos de um formulario inderido em uma "div" como uma classe
  • "form-control" é uma classe pra formação padão de formularios inserido em um "input" como uma classe
  • "placeholder" uma ferramenta para escrever algo no campo do login, alguma instrução

Tabelas

  • "table" cria a estrutura de uma tabela
  • "thead" serve apra colocar o cabeçalho de uma tabela
  • "th" Tabler Headr, complementa o cabeçalho da tabela
  • "tr" Table Row, cria as linhas de uma tebela
  • "td" Table Data, cria as colunar de uma tabela
