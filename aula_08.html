TAGS para modal

BootStrap

  • "modal" serve para definir quando acionado o botão ele chamar uma outra janela ele é uma class no botão tambem é necessario ter uma "id" para fazer a ligação dele no data-target
  • "data-toggle="modal"" esta difinindo que o botao tem função "modal"
  • "data-target="#id"" serve pra quando precionar o botao ele chamar a caixa do modal
  • "modal-dialog" cria a caixa do modal (caixa branca)
  • "modal-content" é div que abre pra colocar dentro dela as funções e textos da caixa
  • "modal-header" cabeçalho da caixa
  • "modal-body" corpo da caixa
  • "modal-footer" roda pé da caixa
  • "close" class do bootstrap para fechar o modal, para funiconar precisa estar dentro de um botao
  • "data-dismiss" class do bootstrap para fechar algum comando nessa caso seria o modal entao a sintax iria ficar "data-dismiss="modal""
  • "modal-titel" serve para colocar um titulo dentro do header
  • "modal fade" faz com que tenha um interação e efeito qunado abre a tela do modal
  • "modal-sm" seria para definir o tamanho do modal de utiliza dentro do "modal-dialog"
