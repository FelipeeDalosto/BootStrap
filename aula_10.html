TAGS para tooltip e popover

HTML

  • tooltip é alerta que mostra quando passa o mouse em cima
  • popover é alerta que mostra quando aciona um botão

BootStrap

  • "page-header" cabeçalho
  • "data-toggle" serve para chamar o tooltip
  • "data-placement" posição para ficao alerta
  • "tilte" oque esta dentro do tooltip
