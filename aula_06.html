CONHECENDO O BOOTSTRAP

TAGS para barra de navegação normal

HTML

  • "nav" serve para crir uma barra de navegação
  • "a href="#"" serve para direcionar a alguma pagina ou arquivo

BootStrap

  • "navbar" cria uma barra de navegação no top da pagina
  • "navbar-default" estilo padrao da barra de navegação
  • "navbra navbar-default" sintax completa dentro dela vai um "container"
  • "navbar-header" cabeçalho da barra de navegação
  • "navbar-brand" onde vai a logo
  • "navbar-nav" formata a lista para que fique igual uma barra de navegação gerante utiliza em um "ul"
  • "navbar-right" coloca a barra para direita geralmente se utiliza em uma "ul"
  • "dropdown" serve para criar um elemente de lista que tenha interações quando clica nele utiliza em um "li"
  • "dropdown-toggle" utiliza se em um link "a href"
  • "data-toggle" serve para criar um vinculo com o "dropdown"
  • "data-toogle="dropdown"" sintax completa
  • "dropdown-menu" formara para que a lista se torne interativa
  • "collapsed" serve quando a pagina fica fora da formação correta
  • navbar-collapsed"
