TAGS para tabelas

HTML

  • "table" cria a tabela.
  • "thead" cria um cabeçalho pra uma tabela.
  • "tbody" Cria o corpo da tabela
  • "tr" Table Row cria as linhas da tabela
  • "th" Table header colocar informações no baçalho
  • "td" Table Data cria as colunas de uma tabela
  • A estrutura é "table" "tr" "td" "/td" "/tr" "table"
  • "colspan" mescla linha
  • "rowspan" mescla colunas

BootStrap

  • Todos comando do bootstrap são inderiso dentro de um class
  • "table" formata a tabala no padrao do bootstrap
  • "table-striped" faz a linhas terem cores diferens
  • "table-bordered" cria uma borda padrao
  • "table-over" faz uma interação quando passa o mouse
  • "table-condensed" define um padding pre definido
  • "sucess" colore o background de verde pode ser usando em "td e "tr"
  • "danger" colore o background de vermelho pode ser usando em "td e "tr"
  • "info" colore o background de azul pode ser usando em "td e "tr"
  • "warnig" colore o background de amarelo pode ser usando em "td e "tr"
